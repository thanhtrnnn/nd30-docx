# Sơ đồ Bố trí Thành phần Thể thức Văn bản Hành chính

Theo Mục IV, Phần I, Phụ lục I – Nghị định 30/2020/NĐ-CP.

---

## 1. Bảng vị trí các ô

| Ô | Thành phần thể thức |
|---|---------------------|
| 1 | Quốc hiệu và Tiêu ngữ |
| 2 | Tên cơ quan, tổ chức ban hành văn bản |
| 3 | Số, ký hiệu của văn bản |
| 4 | Địa danh và thời gian ban hành văn bản |
| 5a | Tên loại và trích yếu nội dung văn bản (VB có tên loại) |
| 5b | Trích yếu nội dung công văn (V/v) |
| 6 | Nội dung văn bản |
| 7a | Quyền hạn, chức vụ của người ký |
| 7b | Họ tên người ký |
| 7c | Chữ ký |
| 8 | Dấu, chữ ký số của cơ quan, tổ chức |
| 9a | Kính gửi (Tờ trình, Báo cáo gửi cấp trên, Công văn) |
| 9b | Nơi nhận |
| 10a | Dấu chỉ độ mật |
| 10b | Dấu chỉ mức độ khẩn |
| 11 | Chỉ dẫn về phạm vi lưu hành |
| 12 | Ký hiệu người soạn thảo và số lượng bản phát hành |
| 13 | Địa chỉ, thư điện tử, trang web, điện thoại, fax |
| 14 | Chữ ký số cho bản sao điện tử |

## 3. Quy tắc bố trí bảng 2 cột

Trong thực tế, phần đầu văn bản (Ô 1-4, 5b) được tổ chức bằng **bảng 2 cột
không viền**:

```
┌─────────────────────┬─────────────────────────────┐
│     CỘT TRÁI        │        CỘT PHẢI             │
│  (chiếm ~45-50%)     │     (chiếm ~50-55%)          │
├─────────────────────┼─────────────────────────────┤
│  Ô 2: Tên cơ quan   │  Ô 1: Quốc hiệu + Tiêu ngữ │
├─────────────────────┼─────────────────────────────┤
│  Ô 3: Số, ký hiệu   │  Ô 4: Địa danh + ngày tháng │
│  Ô 5b: V/v (nếu CV) │                              │
└─────────────────────┴─────────────────────────────┘
```

Phần cuối văn bản (Ô 9b, 7, 8, 12) cũng dùng **bảng 2 cột không viền**:

```
┌─────────────────────┬─────────────────────────────┐
│     CỘT TRÁI        │        CỘT PHẢI             │
├─────────────────────┼─────────────────────────────┤
│  Ô 9b: Nơi nhận     │  Ô 7a: Quyền hạn, chức vụ   │
│  (sát lề trái)       │  Ô 7c: Chữ ký               │
│                      │  Ô 7b: Họ tên               │
│  Ô 12: Ký hiệu      │  Ô 8: Dấu (trùm 1/3 ký)    │
│  người soạn thảo     │                              │
└─────────────────────┴─────────────────────────────┘
```
